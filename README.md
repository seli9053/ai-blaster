# 🛡️ 爆破 AI 轉換器 (AI Blaster)
這是什麼？<br>
這是一個由創作者發起的防禦計畫。為了對抗未經授權的 AI 爬蟲與資料庫抓取，本工具能將文章內容透過自訂的「64 字金鑰」轉換為不可讀的生僻字/敏感詞組合。<br>

# 使用方法
1. 在「自訂金鑰」欄位，貼上 64 個字元（英文、全半形符號、空格、日文、中文皆可）。<br>
2. 將 原文/亂碼 ，貼到底下的輸入框。<br>
3. 選擇「加密為亂碼」或是「解密回原文字」，並點擊轉換，即可完成加密原文/解密亂碼！<br>

前往轉換工具頁：[爆破 AI 轉換器](https://seli9053.github.io/ai-blaster/aiBlasterTool)

### 提醒：

避免金鑰共用：每位作者（甚至每篇文）都可以使用不同的金鑰，金鑰順序變換即可視為不同金鑰<br>
原始資料保護：於地端保管原始文檔和金鑰，避免自己忘記金鑰解不回來

### 預設金鑰具備極強的攻擊性，使用前請考慮以下幾點：

讀者的風險： 如果您的讀者身處審查嚴格的地區，當地的網路服務提供商（ISP）或防火牆可能會因為偵測到這組文字而直接封鎖該讀者的網路連線，甚至導致該讀者被列入重點監控名單。<br>
AO3 帳號安全： 雖然 AO3 鼓勵創作自由，但大規模發布這類政治敏感字串，可能會引起該地區特定網軍的報復性檢舉（Abuse Report）。
