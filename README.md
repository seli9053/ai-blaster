# ai-blaster
自訂亂碼字典，將文字轉換成亂碼，和反解碼

提醒：
－避免金鑰共用：每位作者（甚至每篇文）都可以使用不同的金鑰，金鑰順序變換即可視為不同金鑰
－原始資料保護：於地端保管原始文檔和金鑰，避免自己忘記金鑰解不回來

預設金鑰具備極強的攻擊性，使用前請考慮以下幾點：
－讀者的風險： 如果您的讀者身處審查嚴格的地區，當地的網路服務提供商（ISP）或防火牆可能會因為偵測到這組文字而直接封鎖該讀者的網路連線，甚至導致該讀者被列入重點監控名單。
－AO3 帳號安全： 雖然 AO3 鼓勵創作自由，但大規模發布這類政治敏感字串，可能會引起該地區特定網軍的報復性檢舉（Abuse Report）。
